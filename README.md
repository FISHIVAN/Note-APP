Note Origin - 智能 AI 笔记与旅行助手

Note Origin 是一款基于 Android 平台的现代化笔记应用，深度集成了 AI 智能助手 (Ivan) 和高德地图服务。它不仅是一个记录灵感的工具，更是您的个人知识库和旅行规划伴侣。应用采用 Kotlin 和 Jetpack Compose 构建，提供流畅、美观的用户体验。

核心功能

1. 富文本笔记 (Rich Text Notes)
   支持创建和编辑富文本笔记。
   格式支持：粗体、斜体、标题、列表、文本颜色、高亮颜色等。
   视图切换：支持列表视图和网格视图，方便浏览。
   排序与搜索：支持按标题、创建时间、修改时间排序，并提供全文搜索功能。
2. 待办事项 (Todos)
   快速添加和管理待办任务。
   支持设置截止日期 (Deadline) 提醒。
   直观的完成状态切换与删除确认。
3. AI 助手 Ivan (AI Assistant)
   人格化交互：内置 AI 助手 "Ivan"，拥有温柔俏皮的人格设定，像朋友一样与您对话。
   自然语言操作：通过对话即可创建笔记、添加待办。例如：“帮我记一下明天去买菜” -> 自动生成待办。
   上下文感知：AI 可以读取您现有的笔记和待办（经过摘要处理），提供更贴切的建议。
   智能摘要：支持后台自动生成笔记摘要，节省 AI Token 消耗。
   多模型支持：支持配置 SiliconFlow API，可切换不同的大语言模型。
4. 地图笔记 (Map Notes)
   高德地图集成：内置完整的 3D 地图功能。
   位置标记：在地图上搜索地点并添加自定义颜色的标记 (Marker)。
   AI 联动旅行规划：当您询问旅行攻略（如“南京攻略”）时，AI 会自动在地图上生成对应的景点标记和游玩建议，实现“所聊即所得”的旅行规划体验。
   位置追踪：支持定位当前位置。
5. 个性化设置 (Settings)
   功能开关：可自定义开启/关闭 AI 助手或地图笔记入口。
   API 配置：灵活配置 AI 模型的 API Key 和模型名称。
   主题适配：完美适配系统的深色/浅色模式。
技术架构

开发语言: Kotlin
UI 框架: Jetpack Compose (Material Design 3)
架构模式: MVVM (Model-View-ViewModel)
异步处理: Coroutines & Flow
本地存储: Room Database (SQLite)
网络请求: OkHttp
地图服务: 高德地图 SDK (AMap SDK)

项目亮点

所见即所得的 AI 操作: 独特的 Action 指令系统，让 AI 输出不仅限于文本，更能直接驱动 App 行为（增删改查、地图打点）。
隐私优先: 所有笔记和待办数据存储在本地数据库，AI 请求仅发送必要的摘要信息。
现代 UI: 全面使用 Compose 构建，动画流畅，交互自然。
